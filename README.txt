CustomPrint Estimator lets customers customize print area dimensions and calculates price based on size and printing options for WooCommerce products.


== Description ==

CustomPrint Estimator is a powerful WooCommerce extension that allows customers to customize their product’s print area by entering the width and height on the product page. The plugin calculates the price based on the area and adjusts for single or double-sided printing options. Easily integrate custom pricing logic into your store and offer a seamless pricing experience for print-based products.



== Installation ==

1. Upload mts-customprint-estimator plugin to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How does the CustomPrint Estimator work? =

The estimator allows customers to enter the width and height of their print area, and it automatically calculates the price based on size and printing options.

= Can I offer single and double-sided printing options? =

Yes, the plugin lets customers choose between single-sided or double-sided printing, and the price is adjusted accordingly.

= Is it easy to integrate with my WooCommerce store? =

Yes, the CustomPrint Estimator integrates seamlessly with your WooCommerce store and works directly on the product page.

= Does the plugin calculate the area automatically? =

Yes, the plugin calculates the total print area based on the width and height provided by the customer, and updates the price accordingly.

= Is the plugin mobile-friendly? =

Yes, the CustomPrint Estimator is fully responsive and works seamlessly on mobile devices.
